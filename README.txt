Open NAPLEX2026_StudyHub.html to reach every quiz from one page.
Keep ALL files together in this folder so the hub links work.

USE OFFLINE (no internet):
  - Double-click NAPLEX2026_StudyHub.html (or any single quiz .html).
  - Works on any computer; on phones, AirDrop/email a file and open it.

GET A SHAREABLE LINK (recommended) — pick ONE host, all free:

1) Netlify Drop  (fastest, ~2 min)
   - Go to:  https://app.netlify.com/drop
   - Drag THIS WHOLE FOLDER onto the page.
   - It returns a public https link, e.g. https://your-name.netlify.app
   - Your hub = that link + /NAPLEX2026_StudyHub.html
   - Make a free account to keep the link permanent + rename the site.

2) Cloudflare Pages  (pages.cloudflare.com)  or  GitHub Pages
   - Upload the folder; you get a permanent free https link. A bit more setup.

3) Vercel (vercel.com) — drag-and-drop a folder, instant link.

AVOID Google Drive / Dropbox / iCloud for this — they download or show the
HTML as code instead of running it. Use a static host above.

PUT LINKS IN YOUR STUDY GUIDES:
  Each Q-Bank doc has a "Practice quiz" line with a blank — paste your hosted
  URL there (e.g. https://your-name.netlify.app/NAPLEX2026_Diabetes_QuizApp.html).

NOTE: progress (scores, attempt history, "retest my missed") is saved per
browser/device — each device keeps its own; the quizzes work the same everywhere.
